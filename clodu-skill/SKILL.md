# Song Recommendation Skill

사용자의 기분, 상황, 장르, 아티스트 취향에 맞는 노래를 추천하는 스킬입니다.
단순한 추천을 넘어, 사용자가 왜 이 노래가 자신에게 맞는지 느낄 수 있도록 맥락 있는 추천을 제공합니다.

---

## 트리거 조건 (When to Use)

다음 상황에서 반드시 이 스킬을 사용하세요:

- 직접적인 노래/음악 추천 요청: "노래 추천해줘", "음악 추천", "song recommendation"
- 기분 기반 요청: "슬플 때 들을 노래", "신나는 노래", "잔잔한 음악", "힐링 음악"
- 상황 기반 요청: "드라이브 음악", "공부할 때 음악", "운동할 때 노래", "잠들기 전 음악"
- 장르 기반 요청: "재즈 추천", "R&B 추천", "K-pop 추천", "인디 음악", "클래식"
- 아티스트 기반 요청: "아이유 스타일 노래", "BTS 좋아하는데 비슷한 거", "뉴진스 느낌"
- 플레이리스트 요청: "카페 플레이리스트", "새벽 감성 플레이리스트"
- 암묵적 요청: "요즘 들을 노래 없나", "음악 추천 좀", "뭐 들어?"

## 추천 프로세스 (Step-by-Step)

### Step 1. 사용자 의도 파악

요청에서 다음 정보를 추출하세요:

| 항목 | 예시 |
|------|------|
| **기분/감정** | 슬픔, 설렘, 신남, 차분함, 그리움 |
| **상황/장소** | 드라이브, 공부, 운동, 카페, 새벽, 파티 |
| **장르** | K-pop, 재즈, 인디, R&B, 팝, 클래식, 힙합 |
| **아티스트 취향** | 언급된 아티스트 또는 스타일 |
| **언어 선호** | 한국어 노래, 영어 노래, 무관 |
| **추천 수** | 명시되지 않으면 기본 5곡 |

정보가 부족하면 **한 가지 핵심 질문**만 추가로 묻고, 너무 많은 질문은 피하세요.

---

### Step 2. 추천 노래 선정 기준

- 요청의 맥락(기분/상황/장르)에 **정확히 부합**하는 곡 선정
- 너무 유명한 곡만 나열하지 말고, **숨겨진 명곡(hidden gem)**도 1~2곡 포함
- 아티스트가 겹치지 않도록 **다양성** 확보
- 최신 곡과 클래식(오래된 명곡)을 **균형 있게** 섞기
- 한국어 요청이면 **K-pop/국내 음악 위주**로, 영어 요청이면 **팝/해외 음악 위주**로

---

### Step 3. 추천 출력 포맷

아래 형식으로 출력하세요:

```
🎵 [곡 제목] — [아티스트]
   장르: [장르 태그]
   추천 이유: [사용자의 요청 맥락에 맞춘 1~2줄 이유]
```

**예시:**
```
🎵 Blueming — 아이유 (IU)
   장르: K-pop / 인디팝
   추천 이유: 설레는 감정을 경쾌하고 따뜻하게 담아낸 곡으로, 봄날 드라이브에 딱 어울려요.

🎵 Gravity — John Mayer
   장르: 소프트 록 / 팝
   추천 이유: 잔잔하면서도 깊이 있는 기타 사운드가 드라이브의 여운을 더해줘요.
```

---

### Step 4. 마무리 멘트

추천 목록 이후에 다음 중 하나를 자연스럽게 덧붙이세요:

- 취향이나 분위기에 더 특화된 추천을 원하면 알려달라는 안내
- 플레이리스트 순서 제안 (선택)
- Spotify, Apple Music, YouTube Music 등에서 검색하면 쉽게 찾을 수 있다는 안내 (선택)

---

## 출력 가이드라인

| 항목 | 기준 |
|------|------|
| **기본 추천 곡 수** | 5곡 (사용자가 다르게 요청하면 그에 맞춤) |
| **추천 이유 길이** | 1~2줄, 간결하고 공감 가능하게 |
| **말투** | 사용자의 언어(한국어/영어)에 맞춤, 친근하고 따뜻하게 |
| **형식** | 이모지(🎵) 사용, 가독성 있게 구분 |
| **다양성** | 아티스트 중복 최소화, 장르 편중 방지 |

---

## 엣지 케이스 처리

| 상황 | 대응 방법 |
|------|-----------|
| 요청이 너무 모호할 때 ("그냥 좋은 노래") | 기분이나 상황을 한 가지만 물어보고 추천 |
| 매우 마이너한 장르 요청 | 해당 장르 특성을 간단히 설명하고 대표곡 추천 |
| 특정 아티스트의 비슷한 곡 요청 | 해당 아티스트의 음악 스타일을 분석해 유사 아티스트 추천 |
| 이미 추천한 곡이 마음에 안 든다고 할 때 | 이유를 파악하고 방향을 조정해 재추천 |
| 플레이리스트 전체 구성 요청 | 10~15곡, 순서(도입→절정→마무리)를 고려해 배치 |

> ...

## 참고사항

- 실시간 차트나 최신 발매 정보가 필요한 경우, 웹 검색 도구를 활용해 최신 정보를 보완하세요.
- 사용자가 특정 스트리밍 플랫폼(Spotify, Apple Music 등)을 언급하면 해당 플랫폼에서 찾기 쉬운 곡 위주로 추천하세요.
- 노래 제목과 아티스트명은 **원어 표기**를 기본으로 하되, 한국 아티스트는 한글 병기를 권장합니다.
